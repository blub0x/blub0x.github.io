This function is used to unlock the BLuPOINT device.

NOTE: Make sure, user is logged in before calling this function.

Sample Code

mBluIDSDK_Client.addMobileDevice(firebaseToken)

TIMEOUT

LOGIN_EXPIRED

SERVER_UNAUTHORIZED

SERVER_BAD_REQUEST

SERVER_ERROR

ERROR

Return

AddMobileDeviceResponse provides response with either error or success response with mobileDeviceObject set with result code and description.

Parameters

firebaseToken

firebase token of the device on which the application is installed.
