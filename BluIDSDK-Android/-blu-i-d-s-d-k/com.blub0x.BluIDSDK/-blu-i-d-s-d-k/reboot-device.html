rebootDevice

suspend fun rebootDevice(deviceID: String, afterSeconds: Int?): RebootResponse
Content copied to clipboard

Reboots connected BluPOINT device after given seconds.

Note: Device should be unlocked before calling this function

Sample Code

if (!isBluPOINTUnlocked){
    println("Device needs to be unlocked")
}
val deviceID = "00:0a:0b:0c:0d:0e"
var rebootDeviceResponse = mBluIDSDK_Client.rebootDevice(deviceID, 3)

Possible Errors

ACCESS_DENIED

DEVICE_API_TIMEOUT

DEVICE_INVALID_RESPONSE

DEVICE_FAILURE

INVALID_INPUT

TIMEOUT

Return

RebootResponse Provides either error response with error not null or success response with data not null.

Parameters

deviceID

Device ID of BluPOINT device to reboot

afterSeconds

Delay in seconds to perform reboot of BluPOINT device
