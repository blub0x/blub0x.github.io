updateDeviceBLETxPowerLevel

suspend fun updateDeviceBLETxPowerLevel(deviceID: String, strength: Int): SetStrengthResponse?
Content copied to clipboard

Sets BLE Tx power of connected BluPOINT device, this will advertise all packets and data packets with this power level

Note: Device should be unlock before calling this function

Sample Code

if (!isBluPOINTUnlocked){
    println("Device needs to be unlocked")
}
val deviceID = "00:0a:0b:0c:0d:0e"
var updateDeviceBLETxPowerLevelResponse = mBluIDSDK_Client.updateDeviceBLETxPowerLevel(deviceID, 5)

Possible Errors

ACCESS_DENIED

DEVICE_BUSY

DEVICE_CONNECT_FAILURE

DEVICE_API_TIMEOUT

DEVICE_INVALID_RESPONSE

DEVICE_COMMUNICATION_ERROR

Return

SetStrengthResponse Provides either error response with error not null or success response with data not null.

Parameters

deviceID

Device ID of BluPOINT device to change transmission strength

strength

The strength to be set for the BluPOINT device
