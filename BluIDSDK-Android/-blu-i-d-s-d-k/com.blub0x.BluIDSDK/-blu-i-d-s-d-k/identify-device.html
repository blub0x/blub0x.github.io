suspend fun identifyDevice(withLED: Device_LED, deviceID: String): BluIDSDKError?
Content copied to clipboard

This function is used to detect the unlocked BluPOINT device by setting the device LED to a particular colour for given amount of time.

Sample Code

if (!isBluPOINTUnlocked){
    println("Device needs to be unlocked")
}
var bluPOINTLED = Device_LED(Device_LED_Color(0, 0xFF.toByte(), 0), 5)
var deviceID = "00:0a:0b:0c:0d:0e"
var identifyDeviceResponse = mBluIDSDK_Client.identifyDevice(bluPOINTLED, deviceID)

Possible Errors

DEVICE_API_TIMEOUT

DEVICE_FAILURE

INVALID_INPUT

TIMEOUT

ACCESS_DENIED

DEVICE_INVALID_RESPONSE

DEVICE_COMMUNICATION_ERROR

DEVICE_CONNECTION_LOST

Return

BluIDSDKError Response either error with error not null or success response with error set as null.

Parameters

withLED

Device_LED is the colour and time to which the LED of the unlocked device is set.

deviceID

Device ID of unlocked BluPOINT device which needs to be identified.
